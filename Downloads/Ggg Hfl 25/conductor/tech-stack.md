# Technology Stack

## Core Technologies
- **Frontend Framework:** React (v18.2.0)
- **Programming Language:** TypeScript
- **Build Tool:** Vite (v6.2.0)

## 3D & Data Visualization
- **3D Rendering:** Three.js (v0.160.0)
- **React 3D Bridge:** React Three Fiber (v8.15.11)
- **3D Utilities:** React Three Drei (v9.88.11)
- **Charts & Graphs:** Chart.js (v4.5.1)

## UI & UX
- **Animations:** Framer Motion (v11.11.11) and Motion (v12.38.0)
- **Icons:** Lucide-React (v1.7.0)
- **Markdown Rendering:** React-Markdown (v8), Remark-GFM (v3)
